[0:00 — INTRODUCTION]

Hi, my name is Thakarvind, and this is Sovereign Patch Agent —
an autonomous security system that detects vulnerabilities in code,
generates fixes, and enables safe human review before merging to production.

I built this for the Google AI Agents Intensive Capstone Project

(pause)


[0:15 — THE PROBLEM]

Every day, developers push code that contains hidden vulnerabilities —
SQL injections, exposed credentials, unpatched CVEs.
By the time a human reviewer catches it, the damage may already be done.

Sovereign Patch Agent solves this with a team of AI agents
that work together — just like a real security team would.


[0:30 — WHAT IT IS]

This is a multi-agent system built on Google's Agent Development Kit,
ADK version 2.2.

It chains together four specialized AI agents, each with a single job,
coordinated by a central orchestrator.

Let me show you the architecture.

(show assets/architecture_diagram.svg)

First, every request enters the SecurityGate.
This node scrubs PII, detects prompt injection attempts,
and enforces production guards.
If anything suspicious is found, the request is stopped immediately.

If the input passes, it moves to the VulnerabilityScanner.
This agent scans code for security flaws using MCP tools —
SQL injection, hardcoded secrets, insecure deserialization.

Next is the PatchGenerator.
It creates targeted fixes and validates them in a sandbox.

Then the CodeReviewer does a final pass —
checking quality, adding to the audit trail,
and pausing for human approval.

(pause)

[1:40 — SECURITY AND MCP]

Every decision is logged to a structured JSON audit log.
The MCP Server provides five tools:
scan_codebase, run_sandbox_test, stage_git_pull_request,
get_vulnerability_db, and get_audit_log.

This makes the system extensible —
you can swap out tools without touching the agent logic.

Who benefits?

Security teams overwhelmed by volume.
Developers who want automated pre-merge checks.
Organisations that need compliance-ready audit trails.

Sovereign Patch Agent doesn't replace your security engineer —
it handles the routine work so they can focus on what matters.

That's Sovereign Patch Agent. Built with Google ADK 2.0,
Gemini 2.5 Flash, and MCP.
