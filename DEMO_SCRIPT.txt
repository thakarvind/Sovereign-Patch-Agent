[0:20 — WHAT IT IS]

This is a multi-agent system built on Google ADK 2.2.
It uses a Workflow graph with four nodes and three specialized LLM agents.

Every request starts at the SecurityGate — the first line of defence.

If the input looks clean, it flows to the Orchestrator,
which coordinates three sub-agents:

VulnerabilityScanner — finds security flaws using MCP tools.
PatchGenerator — creates fixes and validates them in a sandbox.
CodeReviewer — does a final check and asks for your approval.

If anything suspicious enters — injection attempts, exposed credentials —
the SecurityGate blocks it immediately and logs a CRITICAL event.
