Every 39 seconds, a cyberattack happens somewhere in the world.

Software vulnerabilities are the #1 entry point — and they cost companies

The problem? Security tools scan your code, but someone still has to read
the report, write the fix, test it, and get it approved.

What if an AI agent could do all of that — automatically?

========================================
[0:20 — WHAT IT IS]
========================================

(show project banner on screen)

This is Sovereign Patch Agent — a multi-agent security system built with
Google ADK 2.0 and Gemini 2.5 Flash.

It uses 4 specialized AI agents working together in a pipeline:

  1. SecurityGate — blocks attacks, scrubs sensitive data
  2. VulnerabilityScanner — finds security flaws in your code
  3. PatchGenerator — writes fixes and tests them in a sandbox
  4. CodeReviewer — gives final approval before anything goes to production

It's not just scanning — it's scanning, fixing, testing, and reviewing.
All with a human in the loop.

========================================
[0:40 — SHOW THE ARCHITECTURE]
========================================

(show architecture diagram)

Here's how it works:

  User submits code → SecurityGate checks for attacks first.
  If clean → VulnerabilityScanner runs pattern-based detection.
  Finds issues → PatchGenerator creates targeted fixes.
  CodeReviewer validates everything → asks for your approval.

On the right, you see the MCP Server — that's where the 5 security
tools live: code scanning, sandbox testing, PR staging, vulnerability
database, and audit logging.

Every single decision gets logged to an audit trail for compliance.

TEST 3 — Real vulnerability:

  PatchGenerator then creates the secure fix automatically.
